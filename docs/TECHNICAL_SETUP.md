# Technical Setup (SkyNeedle)

## Prereqs
- Node.js (LTS)
- npm

## Install
```bash
npm install
```

## T0.1 Visual checkpoint
Run:
```bash
npm run dev
```

Pass criteria:
- A browser tab opens (or you navigate to the printed local URL)
- Screen is **black**
- Centered text reads: **SkyNeedle**
- No console errors
